# Issue Tracker: GitHub

Issues and PRDs for this repository live in GitHub Issues in the canonical
`paulchrisluke/krabiclaw` repository. Use the `gh` CLI with
`--repo paulchrisluke/krabiclaw` so operations do not target a contributor fork.

## Conventions

- Create an issue with `gh issue create --repo paulchrisluke/krabiclaw`.
- Read an issue with `gh issue view <number> --repo paulchrisluke/krabiclaw --comments`.
- List issues with `gh issue list --repo paulchrisluke/krabiclaw` and appropriate label and state filters.
- Comment with `gh issue comment <number> --repo paulchrisluke/krabiclaw`.
- Apply or remove labels with `gh issue edit <number> --repo paulchrisluke/krabiclaw`.
- Close with `gh issue close <number> --repo paulchrisluke/krabiclaw`.

When a skill says to publish a PRD or issue, create it in this canonical repository.
