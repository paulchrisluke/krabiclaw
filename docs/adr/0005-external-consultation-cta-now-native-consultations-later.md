# External Consultation CTA Now, Native Consultations Later

NCLS currently uses an external booking link and does not need a Calendly integration or native KrabiClaw consultation booking for cutover. KrabiClaw will support consultation CTA settings that can link to an arbitrary external URL now, while keeping the model open for a future native consultation flow rather than overloading restaurant reservations or experience bookings as the public concept.

## Considered Options

- External-link CTA now, future-ready internal mode: matches NCLS and keeps migration scope controlled.
- Native consultation booking now: stronger platform completeness, but expands the issue into booking rules, intake fields, notifications, and dashboard triage.
- External URL only with no internal shape: fastest, but creates avoidable rework when native consultations are added.
